How do I generate tree output? 
Linux / macOS (recommended — most accurate parsing):
    tree -fi -a > tree.out         # one full path per line

Legacy tree format also supported:
    tree -a > tree.out             # ├── └── indented format
    tree /path/to/dir > tree.out   # a specific folder

Windows (PowerShell or CMD):
    tree /f > tree.out             # include files
    (if garbled)  tree /f | Out-File tree.out -Encoding utf8

This app supports the standard tree format (├── └── │).
Windows tree (└─, ├─) is auto-detected too.
